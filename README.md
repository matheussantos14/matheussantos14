# 👨🏻‍💻 Matheus Santos

**`Desenvolvedor .NET Back-end`**

Me chamo Matheus Santos e atuo como desenvolvedor com foco em back-end, construção de APIs, integrações entre sistemas, automação de processos e otimização de rotinas em banco de dados. Tenho experiência no desenvolvimento de soluções corporativas utilizando C#, .NET e SQL Server, sempre buscando transformar processos manuais em fluxos mais eficientes, confiáveis e escaláveis.

Atuo principalmente com desenvolvimento de APIs REST, processamento de dados, criação e manutenção de scripts SQL, integrações entre sistemas e rotinas bancárias CNAB, incluindo geração de arquivos de remessa e leitura de arquivos de retorno. Também venho desenvolvendo experiência em projetos de migração de dados entre bancos, incluindo cenários de migração de PostgreSQL para SQL Server, análise de estrutura, adaptação de scripts e validação de dados.

Tenho grande interesse em evoluir continuamente na área de engenharia de software, aprofundando meus conhecimentos em arquitetura, mensageria, microsserviços, cloud e boas práticas de desenvolvimento. Sou uma pessoa comprometida, analítica e com muita disposição para aprender novas tecnologias e enfrentar desafios técnicos.
